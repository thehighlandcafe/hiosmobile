Unsure on how to update HiOSMobile? Take a look below.

Updating HiOSMobile

  1. First, tap the more_vert icon on the top bar of the main HiOSMobile screen.

  2. Tap the settings Settings option.

  3. Then tap the system_updateUpdates option.

  4. Then tap the system_updateCheck for Updates option.

  5. On our website, tap the button that says Download from GitHub
  6. Once on the GitHub release page for HiOSMobile, scroll down to Assets and tap the file HiOSmobile-v[version number].apk.

  7. Once downloaded, tap on the file name in your notifications, and tap Update on the dialog that appears.
